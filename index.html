Install Kubernetes

Install Multipass

Launch Ubuntu VM

multipass launch --name ubuntu-lts--cpus 4 --mem 8G --disk 20G

Open a shell

multipass shell ubuntu-lts

Install k3s

sudo curl -sfL https://get.k3s.io | sh -

Prepare Kubernetes

Download Entando custom resource definitions

wget -c https://docs.entando.com/custom-resources.tar.gz -O - | tar -xz

Create custom resources

sudo kubectl create -f custom-resources

Create namespace

wget https://docs.entando.com/entando.yaml

Configure access to your cluster

IP=$(hostname -I | awk '{print $1}')
sed -i "s/192.168.64.25/$IP/" entando.yaml

Deploy Entando

Deploy Helm Chart

sudo kubectl create -f entando.yaml
